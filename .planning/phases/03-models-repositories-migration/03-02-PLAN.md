# Plan 03-02: Repository Layer Migration

## 目标

将 user-service 的 9 个 repository 和 admin-service 的 5 个 repository 按 D-04 决策合并为 10 个按域分组的 repository，放置在 `api_service/repositories/` 包下。所有 repository 继承 `api_service.common.infra.db.repository.BaseRepository`，方法签名保持与原服务一致（D-05）。
