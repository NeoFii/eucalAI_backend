<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client→relay endpoint | 不可信请求体 + API Key 跨越此边界 |
| relay→upstream LLM | 内部凭证用于调用上游，响应可能包含恶意内容 |
| stream→client | SSE 流中不能泄露内部信息 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-05 | Spoofing | controllers/relay/*.py | mitigate | require_api_key Depends 强制鉴权，无 key 返回 401 |
| T-07-06 | Information Disclosure | lifecycle/orchestrator.py | mitigate | _handle_upstream_error 中 sanitize 上游 API Key，不暴露 target_info 细节 |
| T-07-07 | Denial of Service | lifecycle/stream.py | mitigate | finalize_stream 在 finally 中执行，asyncio.shield 防止计费丢失 |
| T-07-08 | Tampering | adapters/anthropic_convert.py | mitigate | 输入已经过 Pydantic schema 验证，转换函数不执行外部 I/O |
| T-07-09 | Elevation of Privilege | controllers/relay/models.py | mitigate | allowed_models 过滤确保用户只看到授权模型（D-19） |
| T-07-SC | Tampering | npm/pip/cargo installs | accept | 无新增依赖 |
